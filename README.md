# Courses-API
*Course API using Resful Web Services which can handle all types of Requests(GET,POST,PUT,DELETE)


API URLs for:
1.GET   /courses
2.GET   /course/{courseId}
3.POST  /course
4.PUT   /course
5.DELETE /course/{courseId}

Developed using Spring Boot
ORM Tool : Hibernate
Database : MySQL

****************************
*Rest01 is developed using Jersey
